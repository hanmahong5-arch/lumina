Chain Walk / 链式遍历

buildConversationChain(): reverse walk from leaf, like Git commit chain

function buildConversationChain(

messages, leafUuid

) {

const byUuid = new Map(

messages.map(m => [m.uuid, m])

);

const chain = [];

let cur = byUuid.get(leafUuid);

while (cur) {

chain.unshift(cur);

cur = byUuid.get(cur.parentUuid);

}

return chain;

}

Git Commit Analogy

  • Git: Each commit points to parent commit
  • Here: Each message points to parentUuid
  • Branch: Compact creates a new branch (tombstone old)
  • Walk: Start from leaf, follow parent links

Why Reverse Walk?

  • JSONL may contain orphaned messages from aborted turns
  • Forward scan would include dead branches
  • Reverse walk from leaf guarantees a clean linear chain
  • O(n) with Map lookup, no recursion needed
