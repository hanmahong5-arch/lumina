Design Principles

  • Append-only: Never modify written data
  • Crash-safe: Partial writes are detectable
  • No WAL needed: JSONL is its own log
  • Human-readable: Debug with text editor

Recovery Guarantee

  • Process crash: resume from last complete message
  • Ctrl+C mid-turn: detect and inject continue prompt
  • Network drop: retry engine handles, then persist
  • 100MB+ session: auto-chunk, no memory issues
