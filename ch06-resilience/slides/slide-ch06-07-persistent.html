When Activated?

Heartbeat Mechanism

  • Every 30s during wait: yield status message
  • Prevents UI from appearing frozen
  • Shows elapsed time and retry count
  • Allows user abort at any point

6h Safety Valve

Even in persistent mode, total retry time caps at 6 hours. Prevents infinite loops on permanent failures (account issues, API deprecation).
