Why Claude Code's Resilience is Different: LangGraph 的韧性是"Checkpoint 快照"——回到上一个稳定状态。OpenAI SDK 是"指数退避"——重试有限次后报错。Claude Code 是"无人值守韧性"——25% jitter + Retry-After 解析 + 529 分区熔断 + FallbackTriggeredError tombstone + Persistent Mode（5min 退避 + 30s 心跳 + 6h 安全阀），目标不是"减少报错"，是让 agent 可以整夜无人值守运行。
