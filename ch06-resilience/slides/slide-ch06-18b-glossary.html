CHAPTER 06 · Key Terms

Glossary

Circuit Breaker / 熔断器

Stop retrying after N failures to prevent cascade — 像保险丝，电流过载自动断电保护

Exponential Backoff / 指数退避

Wait time doubles per failure (500ms,1s,2s,4s...) — 像"等一会再试"，每次等更久给服务器喘息

Retry Strategy / 重试策略

Policy for retries: max attempts, delay, error types — 像考试补考规则，几次机会间隔多久

Fallback Model / 后备模型

Alternative model when primary fails or overloads — 像备用发电机，主电源断了自动顶上

Persistent Mode / 持久模式

Keep retrying on overload (529) until capacity frees — 像排队等号，不管多久都等到轮到你

Backpressure / 背压

Slow consumer signals fast producer to slow down — 像高速匝道信号灯，车多了减慢进入

Feature Gate / 功能开关

Runtime enable/disable features without deployment — 像开关灯，不拆电线就能控制功能

Withhold-Recover / 扣留恢复

Suppress errors during auto-recovery attempts — 像飞行员处理小故障，乘客无需知道
