双层状态模型: Session vs Turn

Session-level (QueryEngine) 跨Turn存活

  • mutableMessages — 可变消息数组, O(1) push
  • totalUsage — 累计 token 消耗
  • readFileState — 文件内容缓存
  • permissionDenials — 权限拒绝记录
  • discoveredSkillNames — 技能发现(Turn内清零)

Turn-level (State) 每次continue整体替换

  • messages — 当前轮消息视图
  • autoCompactTracking — 压缩状态跟踪
  • turnCount — 轮次计数器
  • transition — 上次continue原因
  • maxOutputTokensRecoveryCount
  • hasAttemptedReactiveCompact

设计考量: mutable state for O(1) append + memory control via splice at compact_boundary. 牺牲快照换取内存可控性。
