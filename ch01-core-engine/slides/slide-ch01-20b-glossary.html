CHAPTER 01 · Key Terms

Glossary

Agent Loop / Query Loop / 智能体循环

Send→LLM→tool_use→execute→repeat until end_turn — 像PDCA循环，不断迭代直到完成

Turn / 轮次

One API call plus any resulting tool execution — 像对话中一次"你问我答"

AsyncGenerator / 异步生成器

JS construct yielding stream events incrementally — 像传送带，每次yield一个事件按需取用

Stream Event / 流事件

Individual event: text delta, tool_use, thinking, or control — 像电报中每条消息携带一小块信息

Continue / Terminal / 继续与终止

Loop states: Continue (7 reasons) or Terminal (10 reasons) — 像交通灯，绿灯继续红灯停

Withhold-Recover / 扣留恢复

Suppress errors during auto-recovery attempts — 像飞行员处理小故障，乘客无需知道

Dual-Layer State / 双层状态

QueryEngine (async) + AppState (React UI) via message passing — 像后厨管做菜，前台管上菜

stop_reason / 停止原因

Why model stopped: end_turn, max_tokens, or tool_use — 像电话挂断：说完了/时间到/要查资料
