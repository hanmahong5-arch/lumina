流式接收: StreamEvent 类型

message_start

  • usage 初始化
  • input_tokens 计数
  • cache_read / cache_creation 统计

message_delta

  • usage 增量更新
  • stop_reason 检测
  • end_turn / tool_use / max_tokens

message_stop

  • usage 累加到 totalUsage
  • 本轮 token 消耗结算
  • 触发 post-sampling hooks

特殊消息类型

TombstoneMessage

fallback切换时的占位消息, 替换被清空的assistant消息

ToolUseSummaryMessage

工具执行摘要, 异步生成, 不阻塞主循环

AttachmentMessage

记忆/技能附件, 注入到下一轮context中
