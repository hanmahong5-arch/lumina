9种终止条件

blocking_limit

maxTurns 不足以完成 blocking 循环, 强制停止

prompt_too_long

413 PTL 恢复失败 (reactive compact 已尝试), 不可继续

image_error

图片尺寸/格式错误 (ImageSizeError / ImageResizeError)

model_error

不可重试的 API 错误 (非 413, 非 max_tokens)

aborted_streaming

用户中断 (Ctrl+C) 发生在 API 流式阶段

aborted_tools

用户中断发生在工具执行阶段

hook_stopped

Stop hook 返回 'stopped' / 'prevented'

completed

模型 stop_reason === 'end_turn' 且无未处理工具调用

max_turns

turnCount 达到 maxTurns 限制
