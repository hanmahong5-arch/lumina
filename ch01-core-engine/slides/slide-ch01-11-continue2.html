恢复路径 4-5: 输出 Token 管理

max_output_tokens_escalate 升级

检测到 stop_reason === 'max_tokens' 时触发

8K → 64K (ESCALATED_MAX_TOKENS)

  • 单次升级, 不可回退
  • 仅当当前 maxOutputTokens < ESCALATED_MAX_TOKENS 时触发
  • 设置 maxOutputTokensOverride = 64000
  • 不消耗 recovery count

"Resume directly—no apology"

  • 最多 3 次 (MAX_OUTPUT_TOKENS_RECOVERY_LIMIT)
  • 每次 recoveryCount++
  • 超过限制 → 正常返回 completed
  • 注入 user message 引导模型续写
