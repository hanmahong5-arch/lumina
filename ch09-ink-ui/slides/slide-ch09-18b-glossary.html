CHAPTER 09 · Key Terms

Glossary

Streaming / 流式渲染

Real-time SSE delivery rendered char-by-char in terminal — 像直播vs录播，实时看模型"打字"

AsyncGenerator / 异步生成器

JS construct yielding events to drive UI render loop — 像传送带，每次yield一个事件按需渲染

React + Ink / 终端 React 框架

React components rendered to terminal with Box/Text primitives — 像乐高积木在终端里拼装

Terminal UI (TUI) / 终端用户界面

Full UI in terminal with keybindings and themes — 像文字版图形界面，键盘操作的完整应用

REPL / 读取-求值-输出循环

Main interactive screen: type prompts, get responses — 像命令行终端，输入命令看到结果

PromptInput / 提示输入组件

Handles text, slash commands, attachments, typeahead — 像智能输入框，能打字也能自动补全

Messages Component / 消息组件

Renders conversation with role-based message styles — 像聊天窗口，不同角色不同样式

Keybindings / 快捷键绑定

Configurable shortcuts: Esc interrupt, Ctrl+L clear — 像IDE快捷键，熟练使用效率翻倍

Theme System / 主题系统

Customizable color schemes via ThemeProvider context — 像手机深色模式，一键切换配色

AppState / 应用状态

Zustand-based React state for UI, separate from engine — 像前台点餐系统，后厨由QueryEngine处理
