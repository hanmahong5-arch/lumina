Messages组件: 自定义Memo

// Custom memo comparator

const MemoMessage = React.memo(Message,

  (prev, next) => {

    // Skip re-render if same message identity

    if (prev.msg === next.msg) return true;

    // Streaming tool uses: compare by ref

    if (prev.msg.contentBlock ===

        next.msg.contentBlock) return true;

    return false;

  }

);

contentBlock Ref Identity

Streaming tool uses share a mutable contentBlock ref. The memo comparator checks object identity (===), not deep equality. Same ref = same tool = skip render.

Render Strategy

  • Messages array grows append-only during streaming

  • Only the last message re-renders per text_delta batch

  • Historical messages are fully memoized

  • Tool result messages update once on completion
