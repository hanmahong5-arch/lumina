ThemeProvider: 主题系统

Dark Mode

Default theme. Optimized for typical terminal backgrounds. High contrast ANSI colors.

Light Mode

Inverted palette for light terminal backgrounds. Adjusted contrast ratios.

Auto Mode

Detects terminal background via OSC 11 query. Falls back to dark.

Zustand Theme Store

  • Global state via Zustand (not React context)

  • Persisted to user settings

  • OSC 11 watcher for real-time detection

  • ThemePicker component for manual override

Why Zustand over Context?

Theme changes affect every component. Zustand selector pattern prevents full-tree re-renders. Components subscribe only to their used theme properties.
