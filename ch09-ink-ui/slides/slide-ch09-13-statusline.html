StatusLine: 防抖状态栏

// StatusLine.tsx - 300ms debounce pattern

const latestRef = useRef(currentState);

latestRef.current = currentState;

 

const [displayState, setDisplayState] =

  useState(currentState);

 

useEffect(() => {

  const timer = setTimeout(() => {

    setDisplayState(latestRef.current);

  }, 300);

  return () => clearTimeout(timer);

}, [currentState]);

Why 300ms?

Tool executions can complete in <100ms. Without debounce, status bar would flash rapidly between states. 300ms provides stable visual feedback.
