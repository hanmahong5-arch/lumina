AppState: Zustand全局Store

// AppState.tsx - Zustand store definition

const useAppState = create((set) => ({

  // Bridge state

  replBridgeEnabled: false,

  

  // Permission state

  permissionMode: 'default',

  

  // Agent tracking

  runningAgents: [],

  

  // Session metadata

  sessionId: null,

  conversationId: null,

  

  // UI state

  isCompacting: false,

  planMode: false,

}));

Key State Slices

  • replBridgeEnabled - IDE integration active

  • permissionMode - Current permission level

  • runningAgents - Active sub-agent list

  • planMode - Plan-only mode toggle

Cross-Component Access

AppState is accessed across the component tree without prop drilling. StatusLine, PromptInput, Messages, and permission components all read from this store via selectors.

Selector Pattern

// Only re-render when permissionMode changes

const mode = useAppState(s => s.permissionMode);
