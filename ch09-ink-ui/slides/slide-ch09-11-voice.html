// insertTextRef cursor splicing

insertTextRef.current = (text) => {

  const before = value.slice(0, cursor);

  const after = value.slice(cursor);

  setValue(before + text + after);

  setCursor(cursor + text.length);

};

Feature Gate: VOICE_MODE

Voice integration is gated behind feature('VOICE_MODE'). Entire voice subsystem is tree-shaken when disabled.
