// main.tsx - launchRepl() simplified flow

async function launchRepl(config) {

  await performSideEffects(); // env, signals, UDS

  const { render } = await import('ink'); // lazy for DCE

  const app = render(<App config={config} />);

  await app.waitUntilExit();

}

Lazy Require for Dead Code Elimination

  • Ink and React are imported dynamically, not at top-level

  • Non-interactive modes (--print, SDK) never load UI dependencies

  • Reduces cold-start by ~200ms for non-REPL paths
