Typeahead建议系统

Slash Commands

/ → [/compact, /clear, /config, /help,

     /init, /login, /brief, /exit ...]

Trigger: / at position 0, fuzzy match by name

@Mentions

@ → [@agent-name, @file-path ...]

Trigger: @ after whitespace, filters agents + files

File References

src/ → [src/main.tsx, src/query.ts,

       src/tools.ts ...]

Trigger: path-like pattern, async filesystem walk

Suggestion Renderer

  • Each mode has its own renderer component

  • Up/Down arrow navigation

  • Tab to accept, Esc to dismiss

  • Max 8 visible suggestions

  • Debounced file system queries (100ms)
