子Agent成本归属

// Advisor iteration cost tracking

// Each sub-agent reports usage back

function accumulateAdvisorCost(

  parentCost, advisorResult

) {

  // Recursive accumulation

  for (const iter of advisorResult.iterations) {

    parentCost.inputTokens +=

      iter.usage.inputTokens;

    parentCost.outputTokens +=

      iter.usage.outputTokens;

    

    // Nested sub-agents (recursive)

    if (iter.subAgentUsage) {

      accumulateAdvisorCost(

        parentCost, iter.subAgentUsage);

    }

  }

}

getAdvisorUsage()

Extracts sub-agent costs from completed advisor results. Walks the iteration tree recursively, summing all nested costs into the parent session total.

Why Recursive?

Coordinator mode can spawn agents that spawn sub-agents (depth 2+). Each level must bubble costs up to the root session for accurate total display.
