// Settings resolution (simplified)

function resolveSetting(key) {

  // Policy: "first source wins"

  if (policy[key] !== undefined)

    return policy[key];

  

  // CLI flags override config files

  if (flags[key] !== undefined)

    return flags[key];

  

  // Cascade: local > project > user

  return local[key] ?? project[key]

    ?? user[key] ?? DEFAULT[key];

}

"First Source Wins" for Policy

Enterprise policy settings take absolute precedence. Even CLI flags cannot override them. This ensures compliance with organization-wide security requirements (e.g., forced permission modes, model restrictions).
