SettingsJson Schema

// SettingsJson type definition

interface SettingsJson {

  // Model configuration

  model?: string;

  smallModel?: string;

  provider?: string;

  

  // Permissions

  permissions?: PermissionSettings;

  allowedTools?: string[];

  deniedTools?: string[];

  

  // Remote control

  remoteControl?: RemoteControlSettings;

  

  // UI preferences

  theme?: 'dark' | 'light' | 'auto';

  verbose?: boolean;

  

  // Feature toggles

  features?: Record<string, boolean>;

  

  // Billing

  billing?: BillingSettings;

}

Model Settings

  • model - Primary model for queries

  • smallModel - Fast model for classifiers

  • provider - API provider (anthropic/bedrock/vertex)

Permission Settings

  • Tool allowlists and denylists

  • Per-directory permission rules

  • Auto-approve patterns

Feature Toggles

  • Local overrides for GrowthBook gates

  • Debug-only features for development

  • Experimental UI features
