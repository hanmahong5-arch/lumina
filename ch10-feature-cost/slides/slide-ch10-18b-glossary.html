CHAPTER 10 · Key Terms

Glossary

Feature Gate / 功能开关

Runtime enable/disable features without deployment — 像开关灯，不拆电线就能控制功能

Beta Flag / 测试标志

Named flag controlling experimental features via betas — 像餐厅"今日特供"，新菜先试营业

Token Budget / 令牌预算

Allocated token allowance per operation, tracked/enforced — 像每月生活费预算，各有额度

Cost Tracking / 成本追踪

Real-time token & dollar accounting with subagent rollup — 像用电量实时监控，随时知道花了多少

Dead Code Elimination / 死代码消除

Build-time removal of unreachable code behind disabled gates — 像装修后清理，用不到的全部搬走

Analytics / 分析遥测

Usage metrics and feature adoption for improvement — 像汽车仪表盘，实时显示关键数据

Settings System / 设置系统

Three-tier config: global/project/session with cascade — 像CSS层叠优先级，越具体优先级越高

Stats Cache / 统计缓存

In-memory token usage stats from API responses — 像运动手表步数统计，实时累计随时查看

Subagent Cost Rollup / 子智能体成本汇总

Aggregate child agent costs to parent for total view — 像项目总预算报表，各团队开销汇总

Perfetto Tracing / Perfetto 追踪

Chrome-compatible tracing for execution timelines — 像手术录像回放，精确回顾每个操作耗时
