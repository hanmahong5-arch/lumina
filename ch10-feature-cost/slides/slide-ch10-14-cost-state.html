成本状态: 累计追踪

// StoredCostState structure

interface StoredCostState {

  totalSessionCost: number; // USD

  modelUsage: Record<string, ModelUsage>;

}

 

interface ModelUsage {

  inputTokens: number;

  outputTokens: number;

  cacheReadTokens: number;

  cacheWriteTokens: number;

  costUSD: number;

  requestCount: number;

}

// addToTotalSessionCost()

function addToTotalSessionCost(usage) {

  const cost = calculateUSDCost(usage);

  state.totalSessionCost += cost;

  state.modelUsage[model].costUSD += cost;

  state.modelUsage[model].requestCount++;

}

Per-Model Tracking

Each model has separate ModelUsage accumulator. When switching models mid-session (e.g., Sonnet for coding, Haiku for classification), costs are attributed correctly.

Token Categories

  • inputTokens - User messages + system prompt

  • outputTokens - Assistant responses + tool calls

  • cacheReadTokens - Reused cached prompt (90% discount)

  • cacheWriteTokens - First-time cache creation (1.25x)

StatusLine Display

Cost is shown in real-time on StatusLine: $0.08. Updated after each API response. Color changes at thresholds ($1, $5, $10).
