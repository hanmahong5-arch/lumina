/* Feature Gate Pattern in Claude Code vs alternatives */

Claude Code:

  growthbook.eval("cost_display_v2") // local, instant

OpenAI/SDK:

  // no gate — endpoint just works or 429s

Why Claude Code's Approach is Different: OpenAI Usage API 是"先花钱, 后知道"——服务端事后统计。LangSmith/LangFuse 是"看仪表盘"——需要接入、观测、但不会阻止你。Claude Code 是"门控 + 实时拦截"——GrowthBook 功能门在客户端即时求值, costState 流式累加 token 价格, 到预算阈值直接拦截而非事后告警。不是"你花了多少", 而是"你现在能不能花"。
