尾随提取 / Trailing Extraction

Incremental extraction captures knowledge from ongoing conversation turns

pendingContext Buffer

New conversation turns are accumulated in a pendingContext buffer. When the buffer exceeds a threshold, an incremental extraction round triggers automatically.

Incremental Rounds

  • Only process turns since last extraction cursor

  • Cursor persisted in extraction state

  • Avoids re-processing entire conversation

  • Merges with existing memory files

Timeline

Turn 1-5: conversation starts

Turn 6: pendingContext threshold met

Background: extraction round 1 (turns 1-6)

Turn 7-12: conversation continues

Turn 13: threshold met again

Background: extraction round 2 (turns 7-13)

Key benefit: Long conversations don't lose early insights to context window overflow because extraction runs incrementally.
