createAutoMemCanUseTool / 工具沙箱

Fine-grained permission control for the memory extraction agent

Allowed (Read)

  • Read - Read any file in project

  • Grep - Search file contents

  • Glob - Find files by pattern

Allowed (Write - Restricted)

  • Write - Only to .claude/ directory

  • Path validation enforced before write

  • Cannot create files outside memory dir

Blocked

  • Bash - No shell execution

  • WebFetch - No network access

  • MCP tools - No external services

  • AskUser - No user interaction

  • Agent - No sub-agent spawning
