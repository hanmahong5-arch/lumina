只存不可推导信息 / Store Only Non-Derivable Info

The extraction prompt explicitly excludes information recoverable from other sources

Excluded (Derivable)

  • Code patterns visible in source files

  • Architecture recoverable from directory structure

  • Git history and commit messages

  • README / package.json metadata

  • Type definitions and interfaces

  • Test suites and fixtures

// extractMemories/prompts.ts

DO NOT store information that can

be derived by reading the codebase

Included (Non-Derivable)

  • User preferences and communication style

  • Rejected approaches and why

  • Environment-specific workarounds

  • Deployment secrets location (not values)

  • Team conventions not in linter rules

  • Context behind architectural decisions

Design Rationale

Storing derivable info wastes the 25KB budget and creates staleness risk when code changes but memory does not update.
