消息处理三函数 / Message Processing Trio

Three key functions handle message eligibility, title extraction, and control flow

isEligibleBridgeMessage

Gate function

  • Validates message structure

  • Checks session ID match

  • Verifies message type

  • Rejects expired messages

  • Dedup check via BoundedUUIDSet

returns: boolean

extractTitleText

Title derivation

  • Extract display title from msg

  • Truncate to 50 characters

  • Sanitize HTML entities

  • Fallback to session ID

  • Used in UI and logging

returns: string (max 50)

controlRequest

Control plane

  • Handles non-data messages

  • Ping/pong heartbeat

  • Reconnection requests

  • Epoch bump signals

  • Shutdown commands

returns: ControlResponse

Message flow: receive -> isEligibleBridgeMessage() -> controlRequest() | extractTitleText() -> dispatch to handler
