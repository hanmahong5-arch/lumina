ReplBridgeHandle 接口 / Interface

Unified handle abstraction with 6 core methods

interface

ReplBridgeHandle {

send(msg: BridgeMessage):

Promise<void>;

 

receive():

AsyncIterable<BridgeMessage>;

 

close(): Promise<void>;

 

isConnected(): boolean;

 

getSessionId(): string;

 

reconnect(): Promise<void>;

}

send()

Enqueue message for transport. Handles serialization, dedup ID injection, and transport selection.

receive()

AsyncIterable stream of incoming messages. Supports backpressure via async generator pattern.

close()

Graceful shutdown: drain pending, close transport, cleanup resources.

reconnect()

Re-establish transport with backoff. Replays from last seq-num in v2.
