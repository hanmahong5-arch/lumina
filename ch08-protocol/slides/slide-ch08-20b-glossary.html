CHAPTER 08 · Key Terms

Glossary

MCP (Model Context Protocol) / 模型上下文协议

Open standard for connecting LLM applications with external data sources and tools. Supports stdio, SSE, and HTTP transports.

就像 USB 协议——定义了 AI 应用和外部工具之间的"通用接口"。

MCP Transports / MCP 传输层

Three transport modes: stdio (local process), SSE (Server-Sent Events over HTTP), and streamable HTTP (bidirectional).

就像快递方式——同城骑手（stdio）、普通快递（SSE）、专线物流（HTTP）。

Tool Use ID / 工具调用标识

Unique identifier linking a tool_use request block to its corresponding tool_result response, ensuring correct pairing.

就像快递单号——确保寄出的包裹和收到的回执一一对应。

Tool Result / 工具结果

The response returned after executing a tool, containing the output content and success/error status.

就像函数的返回值——调用完成后告诉你结果是什么。

Message UUID / 消息唯一标识

Universally unique identifier assigned to each message, enabling deduplication and conversation chain reconstruction.

就像身份证号——每条消息都有独一无二的编号，防止重复和混淆。

Streaming / 流式传输

Real-time delivery of model responses via Server-Sent Events (SSE), showing partial output as it is generated.

就像直播和录播的区别——streaming 让你实时看到模型"打字"。

parentUuid Chain / 父级链

Linked list of message UUIDs enabling conversation reconstruction: each message points to its parent via parentUuid.

就像拼图——根据每块的编号重新拼出完整画面。
