useReplBridge / React Hook

Client-side bridge management with failure tracking and auto-dismiss

Failure Tracking

  • Count consecutive connection failures

  • After MAX_FAILURES=3, stop retrying

  • Display error notification to user

  • Reset counter on successful connect

Auto-Dismiss

FAILURE_DISMISS = 10s

Error notifications auto-dismiss after 10 seconds. Prevents notification pile-up during unstable connections.

// useReplBridge.tsx

useReplBridge() {

failures: 0

MAX_FAILURES: 3

FAILURE_DISMISS: 10_000ms

 

onConnect: reset(failures)

onError: failures++

if failures >= MAX: giveUp()

}

Reconnection Strategy

On disconnect: attempt reconnect with exponential backoff. If bridge handle supports reconnect(), use seq-num recovery. Otherwise, reinitialize from scratch.
