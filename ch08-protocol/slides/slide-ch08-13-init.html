initReplBridge() / 初始化流程

OAuth authentication, version detection, title derivation, and v1/v2 routing

1

OAuth Authentication

Validate existing OAuth token or trigger login flow. Retrieve user profile and organization context.

2

Version Detection

Check feature gates: is v2 enabled? Is SSE transport available? Fall back to v1 if gates are off.

3

Title Derivation

Generate session title via deriveTitle() (max 50 chars). Optionally use Haiku model for LLM-generated title (15s timeout).

4

v1/v2 Route

Based on version detection, instantiate either HybridTransport (v1) or SSETransport (v2), wrap in ReplBridgeHandle, and return to caller.

// initReplBridge.ts

oauth() -> detectVersion() -> deriveTitle() -> v2Enabled ? SSETransport : HybridTransport -> ReplBridgeHandle
