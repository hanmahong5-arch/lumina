## Text Colors (Hierarchy)

| Level | Color | Use For |
|-------|-------|---------|
| Title | `#00D4AA` | Section headings, major labels |
| Subtitle | `#6C63FF` | Subheadings, secondary labels |
| Body | `#A0AEC0` | Descriptions, annotations |
| Muted | `#718096` | Metadata, minor details |
| On dark fills | `#FFFFFF` | Text inside dark shapes |
| Accent teal | `#00D4AA` | Key function names |
| Accent amber | `#FFB84D` | Warning-related text |
| Accent coral | `#FF6B6B` | Error/deny text |
| Accent green | `#48BB78` | Success/allow text |
