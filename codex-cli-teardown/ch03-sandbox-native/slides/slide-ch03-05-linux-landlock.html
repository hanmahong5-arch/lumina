Linux: Landlock + Bubblewrap — 三层叠加防御

Landlock LSM (5.13+)

内核 filesystem 强制访问控制。

精细限定进程"只能读 cwd 子树、只能写 /tmp 和 cwd"；进程 commit ruleset 后不可更改；sandboxing/src/landlock.rs

Bubblewrap (bwrap)

用户态 namespace 工具（systemd 家族）。

新 mount/PID/user namespace + chroot 到最小化根目录；无需 root（unprivileged user namespace）；sandboxing/src/bwrap.rs

seccomp-bpf

syscall 级别过滤器。

BPF 程序声明哪些 syscall 允许调用；拦截危险的 ptrace、mount、reboot 等；最细粒度的 OS 约束层
