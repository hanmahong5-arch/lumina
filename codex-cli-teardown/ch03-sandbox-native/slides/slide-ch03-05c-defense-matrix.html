纵深防御:每种越界,至少被一层挡住

为什么要三层?因为单层都有盲区。把「攻击者想做的事」对「三道机制」连成一张矩阵就清楚了。

攻击者想做的事

Landlock(文件访问)

Namespace + bwrap(隔离)

seccomp(syscall 过滤)

写系统目录 /etc

✓ 拒绝写

~ mount ns

—

ptrace 偷别的进程内存

—

~ pid ns 限范围

✓ 过滤 ptrace

mount 挂载文件系统

—

✓ mount ns

✓ 过滤 mount

开 socket 对外偷传

—

✓ net ns + 代理

~

读法:每一行都至少有一个 ✓ —— 没有哪种越界能同时绕过三层。这就是「纵深防御」:Landlock 管文件、namespace 管隔离、seccomp 管系统调用,互补盲区。旧内核没有 Landlock?优雅降级到 bwrap + seccomp,而不是直接裸奔。
