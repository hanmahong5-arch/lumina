GoalRuntimeEvent — Goal 的生命周期事件

GoalRuntimeEvent enum

// goals.rs:76（节选，共 10 个 variant）

enum GoalRuntimeEvent {

TurnStarted, // turn 开始，注入 goal 上下文

TurnFinished, // turn 正常完成

TaskAborted, // 被中断（用户 Ctrl+C 等）

MaybeContinueIfIdle, // session 空闲检查点

}

goals.rs:76

  • TurnStarted → 将 objective 注入 system prompt
  • TaskAborted → 记录中断，不触发 continue
  • MaybeContinueIfIdle → 核心：闲置时自动续命

自主循环（Autonomy Loop）

Turn 完成

→

Session 空闲？

→

MaybeContinue IfIdle

→

注入 continuation prompt

→

新 turn 开始

  • 用户无操作时，goal 驱动 agent 自动开始下一 turn
  • 检查：goal 是否已完成？budget 是否耗尽？
  • 如果都否 → 注入 CONTINUATION_PROMPT_TEMPLATE
  • 如果用户同时操作 → 事件冲突，defer 到下次空闲

意义：这是 Codex 从"工具"到"自主代理"的关键机制——agent 不再等待人类 prompt，它会主动推进 goal。
