Token Budget — 自主性的安全阀

Double Option 语义

// goals.rs:38, goals.rs:65

token_budget:

Option<Option<i64>>

None // 未设置

Some(None) // 显式无限制

Some(Some(n)) // n 个 token

双层 Option 区分"未指定"与"明确无限"——两个不同的用户意图，不能混用单层 Option。

Budget 消耗阶段

  • 正常阶段 (0–80%)
  • agent 正常运行，goal 注入 system prompt
  • 警告阶段 (80–100%)
  • continuation prompt 追加警告文本
  • agent 被引导向完成而非探索
  • 触限阶段 (100%)
  • 注入 BUDGET_LIMIT_PROMPT_TEMPLATE
  • agent 被指示生成完成摘要并停止

BudgetLimitSteering

enum BudgetLimitSteering {

Allowed, // 正常续

Suppressed, // 强制停

}

  • Allowed → 生成 continuation prompt，触发新 turn
  • Suppressed → 跳过 MaybeContinueIfIdle，goal 结束
  • 确定性有限状态机，无歧义

关键洞察：Token budget 是 Codex 自主性与用户控制之间的唯一可调旋钮。没有 budget，agent 可以无限运行直到 API 账单耗尽。这是把"无限循环"变成"有界自主"的核心机制。
