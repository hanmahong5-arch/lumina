18 个具体 Handler — core/src/tools/handlers/ 全览

OS 执行（3）

  • shell.rs — bash/sh/powershell
  • unified_exec.rs — shell + 沙箱配置统一入口
  • apply_patch.rs — 文件 patch 应用

MCP（2）

  • mcp.rs — 调用外部 MCP server 工具
  • mcp_resource.rs — 读取 MCP server 资源

输入处理（2）

  • view_image.rs — 图像编码为 vision input
  • list_dir.rs — 列目录内容

Agent 编排（4）

  • agent_jobs.rs — 批量子 agent（CSV map-reduce，ch04）
  • multi_agents.rs — v1 多 agent 系统
  • multi_agents_v2/ — v2 长期 inter-agent messaging
  • goal.rs — Goal 创建/更新/查询（ch04）

用户交互（2）

  • request_user_input.rs — 暂停等用户输入
  • request_permissions.rs — 申请额外 sandbox 权限

Meta-tooling（3）

  • plan.rs — 结构化任务列表（对比：Claude Code 的 TodoWrite/TodoRead 两个工具）
  • tool_search.rs — 搜索可用工具
  • tool_suggest.rs — 推荐工具（LLM 发现工具）

扩展 & 降级（2）

  • dynamic.rs — 用户运行时定义工具
  • unavailable_tool.rs — 不可用工具的 stub

设计哲学对比：Claude Code ~30+ 原子工具（小而多，组合靠 LLM）。Codex 18 个大而合的工具（功能打包，减少 LLM 调度次数）。
