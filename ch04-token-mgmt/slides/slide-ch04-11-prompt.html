9 段摘要模板 Summary Template

结构化模板约束 LLM 输出，避免自由发挥导致信息遗漏

#1

Primary Request

用户的核心需求是什么

#2

Key Concepts

涉及的关键技术概念

#3

Files Modified

已修改/创建的文件清单

#4

Errors Encountered

遇到的错误及解决方案

#5

Problem Solving

问题解决思路和决策

#6

User Messages

用户的补充要求和反馈

#7

Tasks

已完成和待完成的任务

#8

Current Work

当前正在进行的工作

#9

Next Step

下一步应该做什么

为什么不让 LLM 自由总结?

自由总结容易遗漏错误记录(#4)和用户反馈(#6)。9 段模板是 checklist，确保关键信息不丢失。prompt.ts 中定义。
