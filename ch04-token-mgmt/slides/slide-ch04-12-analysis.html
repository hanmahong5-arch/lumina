// 9-section summary follows

# Primary Request

// formatCompactSummary

function formatCompactSummary(raw) {

// Strip analysis section

const cleaned = raw.replace(

/<analysis>[\s\S]*?<\/analysis>/,

''

);

return cleaned.trim();

}

为什么需要草稿区?

LLM 在 analysis 标签内可以自由推理，提升摘要质量。但这些推理内容不保留到最终摘要中，避免浪费 token。

类比：CoT Prompting

与 chain-of-thought 相同思路：让模型先思考再回答。analysis 是"思考过程"，summary 是"最终答案"。
