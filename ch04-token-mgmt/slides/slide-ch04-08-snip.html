// snipCompactIfNeeded()

function snipCompactIfNeeded(

messages, tokenBudget

) {

let tokensFreed = 0;

while (totalTokens > tokenBudget

&& messages.length > minKeep) {

const removed = messages.shift();

tokensFreed += removed.tokens;

}

return { tokensFreed };

}

与 Micro 的区别

Micro: 裁剪工具结果内容(保留消息结构)

Snip: 直接删除整条消息(更激进)

tokensFreed Passthrough

释放的 token 数量向上层传递，用于判断是否需要进入下一级压缩(Collapse/Auto)。

保留约束
