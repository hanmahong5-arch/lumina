function evaluateTimeBasedTrigger(

messages, now

触发条件

cold cache 场景：token 已达一定阈值，且存在超时的工具结果

时间衰减：越早的结果越可能被裁剪

Cold Cache 含义

当 API 缓存未命中时（新的对话前缀），旧结果的保留价值降低。裁剪旧结果不会影响缓存命中率。

零 API 成本

纯本地操作，不调用 LLM，用户无感知。工具结果替换为 "[content truncated]" 标记。
