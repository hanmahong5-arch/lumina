COMPACTABLE_TOOLS 白名单

信息类工具可压缩，确认类工具必须保留

可压缩 (informational)

Read

INFO

文件内容可重新读取

Bash

INFO

命令输出可重新执行获取

Grep

INFO

搜索结果可重新搜索

Glob

INFO

文件列表可重新枚举

核心判据：结果可重现

这些工具的输出是"查询"性质，压缩后模型可以重新调用获取最新结果。

不可压缩 (confirmation)

Edit

CONFIRM

编辑确认必须保留

Write

CONFIRM

写入确认必须保留

Agent

CONFIRM

子代理结果不可重现

AskUser

CONFIRM

用户回答不可重复

核心判据：结果不可重现

这些工具的输出是"确认/变更"性质，压缩后丢失会导致模型重复操作或丢失用户意图。

设计启示

区分 info 与 confirm 是所有压缩策略的基础判据。
