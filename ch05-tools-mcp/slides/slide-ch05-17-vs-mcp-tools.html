Why Different: Claude Code MCP is Cache-Aware, Not Just Connected

大多数 MCP client 只管"能否连上"。Claude Code 的 MCP 系统不仅连接——它还知道 MCP server 配置是 prompt 缓存的一部分，通过 prefix-sorted 静态前缀最大化缓存命中，并在 tool lifecycle 中管理 warmup 和 graceful shutdown。

// Claude Code .claude/settings.json: 多层 MCP 配置
{
  "mcpServers": {
    "filesystem": { "command": "npx", "args": ["@modelcontextprotocol/server-filesystem", "./"] },
    "playwright": { "command": "npx", "args": ["@anthropic/mcp-playwright"] }
  }
}
// → 启动时自动 feature() 注册, 纳入 cache budget 管理, 受 permission 网关约束
