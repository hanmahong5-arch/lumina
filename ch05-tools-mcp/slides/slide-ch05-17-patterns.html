5 个工具工程模式 / Engineering Patterns

Reusable patterns extracted from the tool orchestration engine

1

DCE Gate Control / DCE 门控

Use feature() macros for build-time dead code elimination. Zero runtime cost for disabled tools. Bundler strips entire module trees.

2

Cache-Aware Sorting / 缓存感知排序

Place stable tool definitions first in the API request. Dynamic MCP tools appended after. Maximizes prompt cache hit rate across turns.

3

Declarative Concurrency / 声明式并发

readOnlyHint annotation on tool definitions. Runtime infers safe parallelism. No explicit concurrency graph needed.

4

Deferred Loading / 延迟加载

Send tool names without schemas. Model discovers via ToolSearch. Saves 6K-15K tokens per request for rarely-used tools.

5

Transport-Layer Resilience / 传输层韧性

MCP connections handle session expiry, auth refresh, and asymmetric timeouts transparently. Tool authors never see transport failures.
