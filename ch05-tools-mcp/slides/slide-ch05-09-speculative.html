Timeline

  • t=0ms: Zod validates, fires speculative check
  • t=0ms: Pre-hook runs in parallel
  • t=50-200ms: Classifier result ready
  • t=???ms: User sees permission popup, result already cached

Why "Speculative"?

The classifier may determine the Bash command is safe before the user even sees the permission dialog. If the user has auto-approve rules, the tool executes instantly with no wait.

Bash Safety Categories

  • search: grep, find, rg - always safe
  • read: cat, head, tail - safe
  • list: ls, tree - safe
  • neutral: git status - safe
  • write/delete: rm, mv - needs approval
