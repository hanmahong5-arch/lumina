CHAPTER 05 · Key Terms

Glossary

Tool / 工具

Function agent calls to interact with external world — 像厨师的刀和锅，模型是厨师

MCP / 模型上下文协议

Open standard for LLM-to-tool connections via stdio/SSE/HTTP — 像USB协议，AI和工具的通用接口

MCP Server / MCP 服务器

Service exposing tools/resources via MCP protocol — 像USB设备，插上就能被主机识别使用

Tool Schema / 工具模式

JSON Schema defining tool name, description, params — 像说明书，告诉模型工具怎么用

Tool Pool / 工具池

Centralized registry assembled per-turn with cache sorting — 像工具箱，集中管理按分区排序

Deferred Tool / 延迟工具

Schema fetched on-demand, only name visible initially — 像懒加载图片，滚到才加载

Function Calling / 函数调用

LLM generates structured JSON to request execution — 像语音助手调用APP，"帮我叫个车"

Structured Output / 结构化输出

Force model output to conform to JSON schema — 像填表，按固定格式填写不自由发挥

Skill Tool / 技能工具

Injects domain-specific prompts and workflows on demand — 像专家顾问，带着专业知识随叫随到

Lazy Loading / 懒加载

Defer resource loading until needed to save memory — 像网页懒加载图片，滚到才加载
