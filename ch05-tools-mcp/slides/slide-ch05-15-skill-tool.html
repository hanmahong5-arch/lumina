getAllCommands() Merge

  • Bundled skills (src/skills/bundled/)
  • User skills (~/.claude/skills/)
  • Project skills (.claude/skills/)
  • Merged with dedup by name

Prompt-Based Invocation

Unlike traditional tools with JSON schema, SkillTool injects the skill's markdown body as a system message. The model follows instructions rather than filling parameters.

Trigger Matching

  • Explicit: User types /commit or /review-pr
  • Implicit: Model calls Skill tool with name
  • Auto: Trigger field matches user intent
