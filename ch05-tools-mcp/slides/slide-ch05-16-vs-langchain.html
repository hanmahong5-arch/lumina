Design Philosophy Difference

LangChain 的工具系统是为通用AI编程——"你能注册什么就有什么"。Claude Code 的工具系统是为安全AI编程——"你被允许用什么才有什么"。从 feature() DCE 到 speculative classification 到 readOnlyHint 自动并发，安全不是事后验证，而是一等公民。
