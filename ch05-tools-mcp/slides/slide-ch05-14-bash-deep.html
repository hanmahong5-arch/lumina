Sandbox Integration

  • macOS: Apple Sandbox seatbelt profiles
  • Linux: Docker/namespace isolation
  • Windows: restricted token execution
  • Network access controlled per-command

PROGRESS_THRESHOLD

Commands running > 2000ms trigger progress indicator. Shows elapsed time and partial stdout to user. Prevents "is it stuck?" anxiety.
