并发安全 / Concurrency Control

readOnlyHint + isConcurrencySafe() = read parallel, write serial

Concurrent (Parallel)

  • Read, Grep, Glob
  • WebFetch, WebSearch
  • ToolSearch
  • Agent (read-only sub-tasks)

readOnlyHint: true

Serial (Queued)

  • Edit, Write, MultiEdit
  • Bash, PowerShell
  • NotebookEdit
  • Any MCP tool without hint

readOnlyHint: false / undefined

isConcurrencySafe()

  • Checks readOnlyHint from tool definition
  • MCP tools inherit hint from server response
  • Default: serial (safe fallback)

Why It Matters

Model often issues multiple tool_use blocks in one response. Parallel reads complete 3-5x faster. Serial writes prevent race conditions on file edits.
