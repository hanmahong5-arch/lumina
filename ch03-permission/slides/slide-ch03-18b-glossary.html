CHAPTER 03 · Key Terms

Glossary

Permission Mode / 权限模式

Five trust levels from "always ask" to "bypass all" — 像手机定位权限：始终允许/询问/拒绝

Deny by Default / 默认拒绝

Uncertain actions are rejected rather than allowed — 像机场安检，不确定就不放行

YOLO Classifier / YOLO 分类器

Two-stage LLM classifier: fast 64-token + deep 4096-token — 像法院简易程序和普通程序

Chain-of-Thought (CoT) / 思维链

Model reasons step-by-step for 30%+ accuracy gain — 像"请展示解题步骤"，先想清楚再答

Prompt Injection / 提示注入

Malicious instructions hidden in content to hijack model — 像信件里夹带给邮递员的假指令

Circuit Breaker / 熔断器

Stop retrying after N failures to prevent cascade — 像保险丝，电流过载自动断电保护

Denial Tracking / 拒绝追踪

Auto-fallback after 3 consecutive or 20 total denials — 像信用卡连续输错密码被锁定

Human-in-the-Loop / 人在回路

Pauses for human approval before dangerous actions — 像手术前签字确认，关键操作需人类点头

Speculative Check / 推测性检查

Pre-compute classification in parallel for readiness — 像餐厅预判点菜，提前备菜上菜更快
