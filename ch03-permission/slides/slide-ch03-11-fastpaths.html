3 级快速路径 Fast Paths

在调用 ML 分类器 API 之前的零延迟静态检查，按优先级依次执行

1

acceptEdits 模拟

auto mode 内置 acceptEdits 行为：Edit / Write / NotebookEdit 等文件编辑工具自动放行。与 acceptEdits 模式行为一致，但额外叠加分类器保护。

ALLOW 延迟: 0ms | 覆盖: ~30% of auto-mode calls

2

SAFE_YOLO_ALLOWLISTED_TOOLS (~25 工具)

硬编码白名单，包含 Read / Glob / Grep / TodoWrite / Agent 等无副作用或低风险工具。任何模式下均可直接放行。

ALLOW 延迟: 0ms | 覆盖: ~40% of remaining calls

3

分类器 API 调用

仅 Bash / MCP / 未知工具进入此路径。Stage 1 快速判定，必要时 Stage 2 深度推理。API 不可用时 fail-closed 返回 block。

CLASSIFY 延迟: 0.5-5s | 覆盖: ~30% of all calls

classifierDecision.ts 中定义 | 快速路径命中率 ~70%，仅 30% 需要实际 API 调用 | 大幅降低权限判定延迟和 API 成本
