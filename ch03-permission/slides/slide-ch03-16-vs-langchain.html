Why Claude Code's Permission Model is Different

LangChain 和 OpenAI 的权限模型是"配置式"的——你需要显式声明权限。Claude Code 的权限模型是"推断式"的——speculative classification 自动判断安全操作，rule DSL 支持内容感知匹配，fail-closed breaker 确保故障时不会泄漏。安全不是配置清单，而是推理管线。
