Fast Paths (零延迟)

1. acceptEdits 模拟 - 文件编辑类直接放行

2. SAFE_YOLO_ALLOWLISTED_TOOLS - ~25 个安全工具白名单

3. 用户 allow 规则命中 - glob 模式匹配

ML Classifier (核心)

Stage 1: 快速判定 (max_tokens=64)

Stage 2: 深度思考 (max_tokens=4096)

返回 allow/block/undecided

Headless Hook (可选)

headless 模式下的外部决策回调，CI/CD 集成用。调用外部服务审批权限请求。

User Prompt (最终兜底)

所有自动化路径未能决策时，回退到用户交互确认。提供 Allow / Deny / Always Allow 三选项。

关键设计：ML 分类器失败时 fail-closed（拒绝），而非 fail-open
