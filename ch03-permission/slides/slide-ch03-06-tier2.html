Tier 2: Allow 自动放行层

通过 Tier 1 后，检查是否可自动放行（无需用户确认）

A

bypassPermissions 全放行

模式为 bypass 时，所有通过 Tier 1 的工具自动 allow（safety-check 已在 Tier 1 拦截）

B

toolAlwaysAllowedRule

用户配置的 allow 规则命中，工具+内容组合匹配 glob 模式

C

acceptEdits 模拟放行

acceptEdits 模式下自动允许 Edit/Write/NotebookEdit 工具

D

passthrough 跳过

无需权限的工具（如 Read, Glob）直接通过，不进入 Tier 3

// Tier 2 allow logic

if (mode === 'bypassPermissions') {

return { type: 'allowed' };

}

if (matchesAllowRule(tool, input)) {

return { type: 'allowed',

rule: matchedRule };

}

if (mode === 'acceptEdits'

&& isEditTool(tool)) {

return { type: 'allowed' };

}

Passthrough 设计

只读工具（Read/Glob/Grep）无副作用，永远不会进入 Ask 层。减少用户干扰，提升交互流畅度。
