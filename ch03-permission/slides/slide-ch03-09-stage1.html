const result = await classifierAPI({

tool_name: tool.name,

// Parse XML response

const action = parseXML(result);

if (action === 'allow') {

}

if (action === 'block') {

return { allowed: false };

}

// undecided → proceed to Stage 2

设计要点

stop_sequences 使模型在输出 </block> 后立即停止，避免生成冗余 token

max_tokens=64 硬限制确保即使 stop 未触发也快速返回

输入截断：长命令仅发送前 N 字符，减少分类延迟

三种输出

allow - 明确安全，自动放行

block - 明确危险，自动拒绝

undecided - 边界 case，进入 Stage 2

解析失败 = block

XML 格式错误、超时、网络异常均视为 block
