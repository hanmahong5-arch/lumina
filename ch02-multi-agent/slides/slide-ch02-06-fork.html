分叉Agent模式 Fork Agent

核心机制

  • 继承父级完整对话上下文 (messages)
  • 通过 buildForkedMessages() 构建字节级相同前缀
  • 所有fork子agent共享 prompt cache
  • 工具池: tools: ['*'] + useExactTools
  • 权限: permissionMode: 'bubble'

Feature Gate

  • 需要 FORK_SUBAGENT feature flag
  • 与 Coordinator模式互斥
  • 非交互式session中禁用
  • 省略 subagent_type 触发隐式fork

反递归保护

isInForkChild() 检测 FORK_BOILERPLATE_TAG 防止fork嵌套
