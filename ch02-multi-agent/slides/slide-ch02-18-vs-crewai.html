对比: CrewAI vs Claude Code

CrewAI

  • 预定义DAG工作流 (Process: sequential/hierarchical)
  • 共享memory (Short-term + Long-term + Entity)
  • 无权限隔离 - 所有agent共享同一工具集
  • 静态角色分配 (Agent.role + Agent.goal)
  • 固定任务依赖图, 编译时确定
  • Python优先, 单进程多线程

Claude Code v2.1.88

  • 动态派生 (Sync/Async/Fork/Remote 4种模式)
  • 隔离上下文 (每agent独立readFileState)
  • 三层权限隔离 (Permission + Tool + Context)
  • 动态角色 (AgentDefinition + 运行时选择)
  • LLM驱动编排, 运行时动态决策
  • TypeScript, 支持跨进程/跨机器
