CHAPTER 02 · Key Terms

Glossary

Multi-Agent / 多智能体

Multiple agents collaborate with own context and tools — 像项目组，架构师带开发者并行工作

Coordinator / 协调者

Distributes tasks to Workers, synthesizes results — 像项目经理，自己不写代码但整合结果

Worker / 工作者

Task-executing agent with restricted tool access — 像团队成员，接受任务、执行、汇报

Subagent (sync/async/fork) / 子智能体

Child agent for subtasks: sync, async, or fork mode — 像外包，把子任务委派出去

Fan-Out / Fan-In / 扇出与扇入

Spawn parallel sub-tasks then merge results — 像分组讨论再汇报，先分头再合并

Tool Isolation / 工具隔离

Restrict subagent tool access to prevent escalation — 像最小权限原则，实习生没有管理员密码

Prompt Cache Sharing / 提示缓存共享

Forked agents reuse parent prefix for ~60%+ cache hits — 像预制菜，常用前菜提前备好

SendMessage / 发送消息

Coordinator tool to dispatch tasks to workers — 像项目经理发工单，写清任务发给工人
