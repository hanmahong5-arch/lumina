对比: AutoGPT vs Claude Code

AutoGPT

  • 单体递归架构: 一个agent循环自我调用
  • 全工具继承: 子任务拥有与父级相同的完整工具集
  • 无隔离: 共享文件系统和内存空间
  • 无限递归风险: 缺乏深度限制
  • 扁平权限: 无权限降级机制
  • 全局状态: 单一workspace, 无并行隔离

Claude Code v2.1.88

  • 多模式派生: 4种spawn模式按需选择
  • 工具白名单: filterToolsForAgent 三层过滤
  • 三层隔离: Permission + Tool + Context
  • 反递归保护: isInForkChild + MAX_AGENT_TURNS
  • 权限只能向上继承: 安全单调性
  • Worktree: git级别文件系统隔离
