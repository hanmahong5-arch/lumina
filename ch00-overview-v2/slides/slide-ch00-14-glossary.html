CHAPTER 00 В· Key Terms

14 / 16

Token / д»ӨзүҢ

Basic text unit LLMs process; ~3/4 English word вҖ” еғҸиҙ§еёҒзҡ„"еҲҶ"пјҢжЁЎеһӢз”Ёе®ғи®ЎйҮҸж–Үжң¬жҲҗжң¬

Context Window / дёҠдёӢж–ҮзӘ—еҸЈ

Max tokens an LLM handles per request (input+output) вҖ” еғҸе·ҘдҪңеҸ°йқўз§ҜпјҢи¶ҠеӨ§иғҪж‘ҠејҖзҡ„ж–Үд»¶и¶ҠеӨҡ

Agent / жҷәиғҪдҪ“

Autonomous LLM entity with tool access for multi-step tasks вҖ” еғҸиғҪиҮӘдё»жҹҘиө„ж–ҷгҖҒеҶҷд»Јз Ғзҡ„е®һд№ з”ҹ

Agent Loop / жҷәиғҪдҪ“еҫӘзҺҜ

SendвҶ’LLMвҶ’tool_useвҶ’executeвҶ’repeat until end_turn вҖ” еғҸPDCAеҫӘзҺҜпјҢдёҚж–ӯиҝӯд»ЈзӣҙеҲ°е®ҢжҲҗ

Tool / е·Ҙе…·

Function the agent calls to interact with external world вҖ” еғҸеҺЁеёҲзҡ„еҲҖе’Ңй”…пјҢжЁЎеһӢжҳҜеҺЁеёҲ

tool_use Block / е·Ҙе…·и°ғз”Ёеқ—

Structured JSON from model requesting tool execution вҖ” еғҸеҢ»з”ҹејҖеӨ„ж–№пјҢиҚҜжҲҝжү§иЎҢй…ҚиҚҜ

Streaming / жөҒејҸиҫ“еҮә

Real-time SSE delivery of partial model responses вҖ” еғҸзӣҙж’ӯvsеҪ•ж’ӯпјҢе®һж—¶зңӢеҲ°жЁЎеһӢ"жү“еӯ—"

stop_reason / еҒңжӯўеҺҹеӣ 

Why model output stopped: end_turn, max_tokens, tool_use вҖ” еғҸз”өиҜқжҢӮж–ӯеҺҹеӣ пјҡиҜҙе®ҢдәҶ/ж—¶й—ҙеҲ°/иҰҒжҹҘиө„ж–ҷ

MCP / жЁЎеһӢдёҠдёӢж–ҮеҚҸи®®

Open standard connecting LLMs with external tools/data вҖ” еғҸUSBеҚҸи®®пјҢAIе’Ңе·Ҙе…·зҡ„"йҖҡз”ЁжҺҘеҸЈ"

Human-in-the-Loop / дәәеңЁеӣһи·Ҝ

System pauses for human approval before risky actions вҖ” еғҸжүӢжңҜеүҚзӯҫеӯ—зЎ®и®ӨпјҢе…ій”®ж“ҚдҪңйңҖдәәзұ»зӮ№еӨҙ
